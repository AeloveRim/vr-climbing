This is the VRClimbing mod - a CommonLib plugin for Skyrim VR.

Rules:
- Use separation of concerns. Each file should have a distinct purpose. No logic goes into plugin.cpp other than forwarding calls to dedicated functions

../reference contains some mods for you to take inspiration and usage reference from
- CommonLibSSE-NG
- higgs

Build scripts:
- build-vrclimbing.ps1 - Development build script
- release-vrclimbing.ps1 - Creates release zip with SKSE/plugins structure
